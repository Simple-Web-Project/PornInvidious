# porninvidious

requires: `wget`, `lua`, `luasocket`, a posix shell


# how to use
```
lua server.lua
```
go to `http://127.0.0.1:8069` (https doesn't work)



optionally pass ip and port as arguments
```
lua server.lua -i 0.0.0.0 -p 80
```
